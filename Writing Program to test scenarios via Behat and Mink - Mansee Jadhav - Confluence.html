    2. Save the file in root directory where behat is installed. In our case it is in C:\Xampp\PHP

    Write your features file and save it in Features folder of your root folder       

      ->>C:\xampp\php\features

    Features file is the one wherein you actually define the steps to be followed for testing.

    Example testfinal.feature

    testfinal.feature

    Feature: Drupal.org search
      In order to find modules on Drupal.org
      As a Drupal user
      I need to be able to use Drupal.org search
      Scenario: Searching for "behat"
        Given I go to "http://drupal.org"
        When I fill in "Search Drupal.org" with "behat"
        And I press "Search"
        Then I should see "Behat Drupal Extension"
